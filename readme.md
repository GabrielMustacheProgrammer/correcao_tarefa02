# ⚖️ Landing Page - Leandro Miranda Advogado

Landing Page desenvolvida utilizando **HTML5**, **CSS3** e **JavaScript puro**, com foco em apresentar os serviços do advogado **Leandro Miranda**, destacando suas áreas de atuação, experiência e facilitando o contato com potenciais clientes.

---

# 📋 Sobre o Projeto

Este projeto consiste em uma Landing Page moderna, elegante e totalmente responsiva para um escritório de advocacia.

O objetivo é transmitir profissionalismo, confiança e credibilidade através de um design minimalista nas cores **preto**, **cinza** e **branco**, proporcionando uma excelente experiência ao usuário.

---

# 🎯 Objetivos

- Apresentar o advogado e sua experiência.
- Exibir as principais áreas de atuação.
- Demonstrar credibilidade através de estatísticas e depoimentos.
- Facilitar o contato de novos clientes.
- Possuir uma interface moderna e responsiva.

---

# 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6)
